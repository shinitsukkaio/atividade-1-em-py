# Atividade 1 - Números Primos

Verifica se um número é primo e mostra seus divisores em cores.

## Como funciona:
- Testa cada número de 1 até o digitado
- Verde = divisor, Amarelo = não divisor
- Conta total de divisores
- Se tem 2 divisores, é primo

## Como usar:
```bash
python atividade.py
```

## Exemplo:
```
Digite um numero: 7
1 2 3 4 5 6 7 
0 numero 7 foi divisivel 2.
divisores de 7: [1, 7]
7 é primo.
```
